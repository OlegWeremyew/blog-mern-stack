# MERN Blog project

## The server is implemented using - Express
### Technologies:
1. **Express**
2. **TypeScript**
3. **JWT**
4. **Mongodb**
5. **Mongoose**

### Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```
